  • លោក ប័រន័រលី បានបង្ហាញនូវមធ្យោបាយងាយស្រួលមួយក្នុងការអានព័ត៌មានតាមរយៈអាសដ្ឋានច្បាស់លាស់ផ្សេងៗគ្នាតាមប្រភេទទិន្នន័យ ដែល អ្នកប្រើប្រាស់នីមួយៗត្រូវការ។អាសយដ្ឋាន ដែលប្រើនោះត្រូវបានគេហៅថា URL។ ដែលមានសណ្ឋានដូចខាងក្រោម៖

    3.World Wide Web (www)

  • Web Server គឺជាប្រភេទកុំព្យូទ័រដ៏ធំដែលអាចផ្ទុកនូវ Web page និង Web site ជាច្រើនហើយដែលអាចធ្វើឲ្យ Web page និង Web site ទាំងនោះអាចយកទៅប្រើ ប្រាស់បាន ដោយអ្នកប្រើប្រាស់តាមរយៈប្រព័ន្ធ Internet ។

  • Web Browser គឺជាកម្មវិធីដែលអាចបកប្រែទិន្នន័យជាកូដអោយក្លាយជាព័ត៌មាន ដែលអាចស្វែងយល់បាន។ Browser មានច្រើនដូចជា Opera, Google Chrom, Safari,........។

    6.1-Tag

  • Tag គឺជាឈ្មោះកំនត់រួចជាស្រេចនៅចន្លោះ "<>" មានតួនាទីសំរាប់ធ្វើអ្វីមួយ ដូចនឹង Function

    <TagName> Content </TagName>
