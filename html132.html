    3.World Wide Web (www)

  • Web Server គឺជាប្រភេទកុំព្យូទ័រដ៏ធំដែលអាចផ្ទុកនូវ Web page និង Web site ជាច្រើនហើយដែលអាចធ្វើឲ្យ Web page និង Web site ទាំងនោះអាចយកទៅប្រើ ប្រាស់បាន ដោយអ្នកប្រើប្រាស់តាមរយៈប្រព័ន្ធ Internet ។

  • Web Browser គឺជាកម្មវិធីដែលអាចបកប្រែទិន្នន័យជាកូដអោយក្លាយជាព័ត៌មាន ដែលអាចស្វែងយល់បាន។ Browser មានច្រើនដូចជា Opera, Google Chrom, Safari,........។

  • Browser: Internet Explorer, Firefox, Netscape, Opera, Saferi, Google Chrome, និង browser ដទៃទៀត។

    5.1-Tag

  • Tag គឺជាឈ្មោះកំនត់រួចជាស្រេចនៅចន្លោះ "<>" មានតួនាទីសំរាប់ធ្វើអ្វីមួយ ដូចនឹង Function

    • Syntax:

    <TagName> Content </TagName>

    <b>Hello word</b>

    <b>well com for you</b>

  • ភាគច្រើន រាល់ Tags នីមួយៗ តែងតែមាន Open Tag និង Close Tage ប៉ុន្តែក៏ មាន Tag មួយចំនួនមិនមាន Close Tag ដែរ ។

    • <area>

    • <base>

    • <br>

    • <col>

    • <command>

    • <Embed>

      • Tag ការបង្ហាញ
      <b>, <strong> ប្រើសម្រាប់អក្សរដ៏រឹងមាំ
      <i> ប្រើសម្រាប់អក្សរជាទ្រីត
      <small> ប្រើសម្រាប់អក្សរតូច
      <pre> 
      ប្រើសម្រាប់អត្ថបទដែលមានទ្រង់ទ្រាយដើម (pre-writing)
      <u> ប្រើសម្រាប់អត្ថបទមានគូសបន្ទាត់ក្រោម
      <del>, <strike> ប្រើសម្រាប់អត្ថបទដែលលុបចោល
